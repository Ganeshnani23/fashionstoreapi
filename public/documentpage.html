Getting Started with the API !

Build faster with FashionStore API - Complete reference and usage examples in one place.

Routes Supported

Get all products

fetch("https://fashionstoreapi-6zko.onrender.com/getallfashionproducts")
.then(res => res.json())
.then(res => console.log(res))

Get a single products

fetch("https://fashionstoreapi-6zko.onrender.com/getfashionproductsbyid/1001")
.then(res => res.json())
.then(res => console.log(res))

replace the product id between 1001 - 1140 for different products

Get categories

fetch("https://fashionstoreapi-6zko.onrender.com/category")
.then(res => res.json())
.then(res => console.log(res))

Get products by cateogories

fetch("https://fashionstoreapi-6zko.onrender.com/fashion/mens_wear")
.then(res => res.json())
.then(res => console.log(res))

replace the mens_wear with womens_wear (or) with kids_wear for different category products
